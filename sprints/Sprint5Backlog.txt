User Stories from Product Backlog that we will address in this sprint:
a) As a user, I can copy file to a new location so that I don't have to delete the old file in some cases
b) As a user, I can see the error message when displaying the content given an incorrect filename so that I can change to the correct filename quickly
c) As a user, I can download file from the internet so that I can communicate with the internet
d) As a user, I can save my workspace to a real computer so that I can come back later
e) As a user, I retrieve my old workspace back so that I can continue my work on JShell
f) As a user, I can find files given patterns so that I can quickly get the path of a filename
g) As a user, I can list the directory structure of the current working directory so that I can have a better concept of what my file system looks like
