User Stories from Product Backlog that we will address in this sprint:
a) As a user, I can see the command history so that I can know what I have done so far
b) As a user, I can see the content of files so that I know what I stored in these files
c) As a user, I can write contents to files so that I can store messages
d) As a user, I can append messages to files so that I can add messages
e) As a user, I can see the usage of each command so that I know how to use the command
